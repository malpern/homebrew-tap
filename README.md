# homebrew-tap

Homebrew formulae for [malpern](https://github.com/malpern)'s tools.

## Install

```bash
brew tap malpern/tap
```

## Formulae

| Formula | Description |
|---------|-------------|
| [barswitch](https://github.com/malpern/barswitch) | Coordinates SketchyBar and the native macOS menu bar |

### BarSwitch

```bash
brew install malpern/tap/barswitch
```
